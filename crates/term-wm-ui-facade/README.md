# term-wm-ui-facade

Internal UI component facade for [term-wm](https://crates.io/crates/term-wm). Not intended for direct use.

Serves as a static, compile-time aggregation layer that unifies [term-wm-ui-components](https://crates.io/crates/term-wm-ui-components) and [term-wm-sys-ui-components](https://crates.io/crates/term-wm-sys-ui-components) behind a single enum boundary.

See the main [term-wm](https://crates.io/crates/term-wm) crate for documentation.

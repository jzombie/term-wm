Where `compute_notification_bounds` is a helper that uses `textwrap::wrap()` to calculate the exact word-boundary-aware height (matching Ratatui's layout), returning a `LayoutRect`:

```rust
fn compute_notification_bounds(msg: &str, area: LayoutRect, index: usize) -> LayoutRect {
    const TOAST_W: u16 = 40;
    const MARGIN: u16 = 2;
    const GAP: u16 = 1;
    let lines = textwrap::wrap(msg, TOAST_W.saturating_sub(2) as usize);
    let h = (lines.len() as u16).saturating_add(2) // + 2 for borders
        .min(area.height.saturating_sub((index as u16) * MARGIN));
    let x = area.width.saturating_sub(TOAST_W).saturating_sub(MARGIN);
    let y = MARGIN + (index as u16) * (h + GAP);
    LayoutRect { x: x as i32, y: y as i32, width: TOAST_W, height: h }
}
```

New method:

```rust
fn render_notification(
    &mut self,
    target_buf: &mut Buffer,
    area: ratatui::layout::Rect,
    _region: &RenderRegion,
    queue: &NotificationQueue,
) {
    // Find notification by scan (small N, 1-5 items)
    let Some(n) = queue.renderable()
        .find(|n| area_belongs_to_notification(area, n))
    else { return };
    let rect = area;
    Clear.render(rect, target_buf);
    let block = Block::default()
        .borders(Borders::ALL)
        .border_style(Style::default().fg(Color::White))
        .style(Style::default().bg(Color::DarkGray).fg(Color::White));
    Paragraph::new(n.message.as_str())
        .block(block)
        .wrap(Wrap { trim: true })
        .render(rect, target_buf);
}
```

## Critical Issues Addressed

| Issue | Solution |
|---|---|
| **No enum on DrawPlan** | `RegionType` field on `RenderRegion` — homogeneity preserved |
| **Event passthrough** | `HitTarget::Notification` dead zone in hitbox dispatch; registered before render |
| **textwrap + unicode** | `textwrap` defaults to `unicode-width` width calculation; no char-count confusion |
| **Dual expiry** | Only `SystemTask::DismissNotification` — no `tick()`/`expires_at` on queue |

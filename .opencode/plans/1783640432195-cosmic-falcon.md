# Notification (Toast) System — Plan

## Overview

Add transient toast notifications rendered as `DrawPlan` regions with hitbox-registered dead zones to swallow mouse events. Timeout via `SystemTask::DismissNotification`.

---

### `crates/term-wm-core/src/draw_plan.rs`

Key moves into the enum variant — illegal state unrepresentable:

```rust
pub enum RegionType {
    Window(WindowKey),    // always has a key
    Notification(Arc<str>), // never needs one
}

pub struct RenderRegion {
    pub bounds: LayoutRect,
    pub z_index: usize,
    pub dimmed: bool,
    pub region_type: RegionType,
}
```

**No `key: Option<WindowKey>` field** — the key lives inside the `Window` variant. The compiler guarantees a `Window` region always has a key and a `Notification` region never does.

### `crates/term-wm-core/src/constants.rs` (or existing constants file)

```rust
pub const NOTIFICATION_Z_INDEX: usize = 20;
```

Centralizes the z-order value; one place to adjust when new layers are added.

## Renderer (draw_plan_renderer.rs)

Destructure key directly from the match — no `.expect()`:

```rust
for region in draw_plan.regions() {
    let area = layout_rect_to_rect(region.bounds);
    match &region.region_type {
        RegionType::Window(key) => {
            let Some(component) = wm.component_for_key_mut(*key) else { continue; };
            if region.z_index < 10 {
                self.render_window_composite(area, component, region, hitbox_registry);
            } else {
                self.render_direct(area, component, region);
            }
        }
        RegionType::Notification(msg) => {
            Clear.render(area, target_buf);
            let block = Block::default()
                .borders(Borders::ALL)
                .border_style(Style::default().fg(Color::White))
                .style(Style::default().bg(Color::DarkGray).fg(Color::White));
            Paragraph::new(msg.as_ref())
                .block(block)
                .wrap(Wrap { trim: true })
                .render(area, target_buf);
        }
    }
}
```

## Layout Engine (engine.rs)

### `generate_notification_regions` (extracted helper)

Early return skips only notification layers, not the pipeline:

```rust
fn generate_notification_regions(plan: &mut DrawPlan, wm: &WindowManager) {
    use textwrap::Options;

    const TOAST_W: u16 = 40;
    const MARGIN: u16 = 2;
    const GAP: u16 = 1;

    let managed = wm.managed_area();

    if managed.width <= MARGIN.saturating_mul(2).saturating_add(2) { return; }

    let actual_w = TOAST_W.min(managed.width.saturating_sub(MARGIN.saturating_mul(2)));
    let inner_w = actual_w.saturating_sub(2) as usize;
    let wrap_opts = Options::new(inner_w).trim(true);

    let mut y_offset: u16 = MARGIN;

    for notification in wm.notifications().renderable().rev() {  // newest first
        let lines = textwrap::wrap(&notification.message, &wrap_opts);
        let h = (lines.len() as u16).saturating_add(2);
        let h = h.min(managed.height.saturating_sub(y_offset.saturating_add(MARGIN)));
        if h < 3 { break; }

        let x = managed.x.saturating_add(managed.width as i32)
            .saturating_sub(actual_w as i32)
            .saturating_sub(MARGIN as i32);

        plan.push(RenderRegion {
            bounds: LayoutRect {
                x,
                y: managed.y.saturating_add(y_offset as i32),
                width: actual_w,
                height: h,
            },
            z_index: NOTIFICATION_Z_INDEX,
            dimmed: false,
            region_type: RegionType::Notification(
                std::sync::Arc::clone(&notification.message),
            ),
        });

        y_offset = y_offset.saturating_add(h).saturating_add(GAP);
    }
}
```

All coordinates anchored to `managed.x` / `managed.y`. All arithmetic uses `saturating_*`.

## Review Compliance Summary

| Flaw | Fix |
|---|---|
| Type-state / panic | `key` inside `RegionType::Window(key)` — compiler guarantees presence; no `.expect()` |
| Magic z_index | `NOTIFICATION_Z_INDEX` constant in central location |
| Heap churn | `Arc<str>` — clone is refcount increment |
| Zero-width crash | Geometric circuit breaker: early return if `width <= 6` |
| Textwrap/Ratatui desync | `textwrap::Options::new(w).trim(true)` matches `Wrap { trim: true }` |
| Dual expiry | Only `SystemTask::DismissNotification` — no `tick()` |
| DrawPlan bypass | Notifications are `RenderRegion`s in `DrawPlan`, rendered by compositor loop |
| Event passthrough | `HitTarget::Notification` dead zone registered in hitbox |
| Coordinate origin | `x`/`y` anchored to `managed.x`/`managed.y` |
| Y-stack overflow | Mutable accumulator, not `index * h` |

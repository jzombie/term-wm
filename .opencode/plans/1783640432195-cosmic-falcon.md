# VerticalStack — Scroll-Aware Layout + Event Routing

## Changes

### 1. `Component` trait — add `desired_height`

Default 0 (stretch to fill).

### 2. `LabelComponent` — render text, desired_height=1

### 3. `ButtonComponent` — render border+label, desired_height=3, uses `localize_mouse_click`

### 4. `VerticalStackComponent` — scroll-aware layout and event routing

**render**: For each child, compute `child_local_area` (local y offset) and `child_screen_area` (absolute y minus scroll_y). Clone context with child's `screen_area`, call `child.render(backend, child_local_area, &child_ctx, registry)`.

**handle_events**: For each child, compute `child_screen_area` the same way. If mouse click intersects child's screen area, clone context with child's `screen_area`, call `child.handle_events(event, &child_ctx)`. Return first non-Ignored result.

**Scroll offset**: Read `ctx.scroll_handle().map(|h| h.info().offset_y as i32)` in both phases.

### 5. `WmSystemPanelComponent` — wrap VerticalStack in ScrollView

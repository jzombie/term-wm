# Notification (Toast) System — Final Plan

## Three Structural Fixes from Review

| Flaw | Fix |
|------|-----|
| **DrawPlan bypass** | Add `DrawPlanLayer` enum; notifications become a proper composited layer |
| **Text wrapping desync** | Use `textwrap::wrap()` (word-boundary aware, matches Ratatui) |
| **Dual expiry** | Remove `tick()`/`expires_at`; rely solely on `SystemTask::DismissNotification` |

### 1. New: `crates/term-wm-core/src/notification.rs`

Pure data — no Ratatui, no rendering logic.

```rust
#[derive(Debug, Clone)]
pub struct Notification {
    pub id: u64,
    pub message: String,
}

#[derive(Debug, Default)]
pub struct NotificationQueue {
    notifications: VecDeque<Notification>,
    next_id: u64,
    max_capacity: usize,  // 5
}

impl NotificationQueue {
    pub fn push(&mut self, message: String) -> u64;        // returns id
    pub fn dismiss(&mut self, id: u64) -> bool;
    pub fn renderable(&self) -> impl Iterator<Item = &Notification>;  // newest first
    pub fn len(&self) -> usize;
}
```

`push` evicts the oldest notification when at `max_capacity`.

### 7. Add rendering: `crates/term-wm-console/src/draw_plan_renderer.rs`

Two changes:

**a) Handle `DrawPlanLayer::Notifications`** in the render loop (`render` / `render_to_buffer`):

```rust

    match layer {
        DrawPlanLayer::Window(region) => { /* existing logic */ }
        DrawPlanLayer::Notifications => {
```

**b) `render_notifications` method:**

Downcasts to `RatatuiBackend` (already native to this crate), uses `textwrap::wrap()` for word-boundary-accurate height pre-calculation, renders `Clear` + `Block` + `Paragraph`:

```rust
fn render_notifications(
    &mut self,
    target_buf: &mut Buffer,
    queue: &NotificationQueue,
    full_area: LayoutRect,
) {
    const TOAST_W: usize = 40;
    const MARGIN: u16 = 2;
    const GAP: u16 = 1;

    let full = layout_rect_to_rect(full_area);
    let mut y = MARGIN;

    for n in queue.renderable() {
        // Word-boundary accurate height via textwrap
        let lines = textwrap::wrap(&n.message, TOAST_W.saturating_sub(2));
        let h = (lines.len() as u16).saturating_add(2)  // + borders
            .min(full.height.saturating_sub(y + MARGIN));
        if h < 3 { break; }

        let x = full.width.saturating_sub(TOAST_W as u16).saturating_sub(MARGIN);
        let rect = Rect { x, y, width: TOAST_W as u16, height: h };

        Clear.render(rect, target_buf);
        let block = Block::default().borders(Borders::ALL)
            .border_style(Style::default().fg(Color::White))
            .style(Style::default().bg(Color::DarkGray).fg(Color::White));
        Paragraph::new(n.message.as_str())
            .block(block)
            .wrap(Wrap { trim: true })
            .render(rect, target_buf);

        y = y.saturating_add(h).saturating_add(GAP);
    }
}
```

# Plan: Command Palette Works EXACTLY Like Help Overlay

## Principle

Zero core trait modifications. Zero new methods. Same storage, same lifecycle, same barrier.

## Reference: Help overlay lifecycle

1. **Init**: Nothing. The component is created on demand.
2. **Open**: `open_help_overlay()` creates `WmHelpOverlayComponent`, calls `wm.open_overlay(OverlayId::Help, Some(Box::new(h)))`. Inserted into `BTreeMap<OverlayId, Box<dyn Overlay>>`.
3. **Close**: `wm.close_help_overlay()` → `self.overlays.remove(&OverlayId::Help)`. Removed from map.
4. **Event barrier**: Runner checks `wm.help_overlay_visible()` (which is `contains_key`). If true, calls `wm.handle_help_event()` and returns early.
5. **Handle events**: `handle_help_event` gets the overlay from map, calls `boxed.handle_events()`, processes actions, removes if no longer visible.

pub fn close_command_palette(&mut self) {
    self.overlays.remove(&OverlayId::CommandPalette);
}

pub fn handle_command_palette_event(&mut self, event: &Event) -> Option<TermWmAction> {
    let palette = self.overlays.get_mut(&OverlayId::CommandPalette)?;
    if let Event::Mouse(mouse) = event {
        self.hover = Some((mouse.column, mouse.row));
    }
    let ctx = self.component_context(false)
        .with_overlay(true)
        .with_screen_area(self.managed_area())
        .with_hover_pos(self.hover);
    match palette.handle_events(event, &ctx) {
        EventResult::Action(action) => {
            self.close_command_palette();
            Some(action)
        }
        _ => None,
    }
}

### Step 6: Update main.rs

**`src/main.rs`** — Implement `open_command_palette`:
```rust
fn open_command_palette(&mut self) {
    use term_wm_sys_ui_components::wm_command_palette::WmCommandPaletteComponent;
    let wm = self.inner.wm();
    let mut palette = WmCommandPaletteComponent::new();
    // Set menu items based on current state
    use term_wm_core::components::ComponentAction;
    let items = wm_menu_items(
        wm.mouse_capture_enabled(),
        wm.clipboard_enabled(),
        wm.window_selection_enabled(),
        wm.window_count() > 0,
    );
    // Filter by supported menu actions
    let supported = wm.supported_menu_actions();
    let items: Vec<_> = items.into_iter()
        .filter(|item| supported.contains(&item.action))
        .collect();
    palette.process_action(&ComponentAction::SetMenuItems(items));
    palette.process_action(&ComponentAction::SetManagedArea(wm.managed_area()));
    palette.set_selection_enabled(wm.clipboard_enabled());
    wm.open_overlay(OverlayId::CommandPalette, Some(Box::new(palette)));
}
```

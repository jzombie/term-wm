# Plan: Global Z-Depth via WindowManager, Clean Render Loop

## Architecture

Move the global z-depth normalization into `WindowManager` so `runner.rs` stays declarative.

## Changes

### 1. `crates/term-wm-core/src/component_context.rs`
Add `z_depth: f32` field:
```rust
pub struct ComponentContext {
    // ... existing fields
    z_depth: f32,  // default 1.0 in new()
}
```
```rust
pub const fn z_depth(&self) -> f32 { self.z_depth }
pub fn with_z_depth(&self, z_depth: f32) -> Self { let mut c = self.clone(); c.z_depth = z_depth; c }
```

### 2. `crates/term-wm-core/src/window/window_manager/mod.rs`
Add helper methods:
```rust
/// Count of overlays that will be rendered this frame.
pub fn visible_overlay_count(&self) -> usize {
    let mut n = 0;
    if self.wm_overlay_visible() { n += 1; }
    if self.overlays.contains_key(&OverlayId::ExitConfirm) { n += 1; }
    if self.overlays.contains_key(&OverlayId::Help) { n += 1; }
    n
}

/// Global z-depth normalized across windows + overlays.
/// position: 0-indexed position in the draw order.
/// total: total items (plan.len() + visible_overlay_count()).
pub fn compute_z_depth(position: usize, total: usize) -> f32 {
    if total <= 1 { return 1.0; }
    position as f32 / (total - 1) as f32
}
```

### 3. `crates/term-wm-core/src/window/window_manager/overlays.rs`
Update `render_overlays` to accept z-offset info and pass z_depth via context:
```rust
pub fn render_overlays(&mut self, frame: &mut UiFrame<'_>, overlay_base: usize, total_f: f32) {
    let mut j = 0;
    // ...
    if self.wm_overlay_visible() {
        let z = Self::compute_z_depth(overlay_base + j, total_f);
        j += 1;
        let menu_ctx = self.component_context(false)
            .with_overlay(true).with_z_depth(z)
            .with_hover_pos(self.hover)
            .with_keybindings(...);
        // ...
    }
    // same pattern for confirm + help overlays
}
```

### 4. `crates/term-wm-core/src/runner.rs` — `draw_window_app`
Replace the inline loop with:
```rust
let plan = app.windows().window_draw_plan(frame);
let total = plan.len() + app.windows().visible_overlay_count();

for (i, task) in plan.into_iter().enumerate() {
    let z = WindowManager::<Id>::compute_z_depth(i, total);
    match task {
        DrawTask::App(mut window) => {
            window.surface.z_depth = z;
            // ... rest unchanged ...
        }
        DrawTask::System(mut window) => {
            window.surface.z_depth = z;
            // ... rest unchanged ...
        }
    }
}
app.windows().render_panel(frame);
app.windows().render_overlays(frame, plan.len(), total);
```

### 5. `crates/term-wm-sys-ui-components/src/wm_help_overlay.rs`
Read z_depth from context instead of hardcoded 1.0:
```rust
term_wm_core::ui::render_drop_shadow(frame, shadow_dest, ctx.z_depth());
```

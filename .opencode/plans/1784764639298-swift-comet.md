# Revised Implementation Plan: Floating/Tiling Window Behavior

## Fixes This Round

1. **Pure preview path**: `simulate_position_based_layout()` returns sorted `Vec<(WindowKey, Rect)>` without touching `self`. Preview cache calls this. Only `apply_snap()` mutates state.
2. **Destructive commit**: `apply_position_based_layout()` clears floating rects and rebuilds `managed_layout`. Called only from `apply_snap()`.
3. **Zero-copy sort**: `calculate_tiling_order()` takes ownership of `Vec`.

## Implementation Plan

### Step 1: `float_all_windows()` — atomic layout drop
**File**: `crates/term-wm-core/src/window/window_manager/drag.rs`

```rust
pub(super) fn float_all_windows(&mut self) {
    let tiled_keys: Vec<_> = self.regions.ids()
        .iter()
        .filter(|key| !self.is_window_floating(**key))
        .copied()
        .collect();

    if tiled_keys.is_empty() {
        return;
    }

    let regions: Vec<_> = tiled_keys.iter()
        .map(|key| (*key, self.region(*key)))
        .collect();

    self.managed_layout = None;

    for (key, rect) in regions {
        self.set_floating_rect(
            key,
            Some(crate::window::FloatRectSpec::Absolute(
                crate::window::FloatRect {
                    x: rect.x,
                    y: rect.y,
                    width: rect.width.max(1),
                    height: rect.height.max(1),
                },
            )),
        );
    }
}
```

### Step 3: Zero-copy Euclidean sort (takes ownership)
**File**: `crates/term-wm-core/src/window/window_manager/drag.rs`

```rust
fn calculate_tiling_order(
    mut floating_rects: Vec<(WindowKey, Rect)>,
    target_rect: Rect,
) -> Vec<(WindowKey, Rect)> {
    let target_center_y = (target_rect.y as i32) + (target_rect.height as i32) / 2;
    let target_center_x = (target_rect.x as i32) + (target_rect.width as i32) / 2;

    floating_rects.sort_by(|a, b| {
        let a_dy = ((a.1.y as i32) + (a.1.height as i32) / 2) - target_center_y;
        let a_dx = ((a.1.x as i32) + (a.1.width as i32) / 2) - target_center_x;
        let b_dy = ((b.1.y as i32) + (b.1.height as i32) / 2) - target_center_y;
        let b_dx = ((b.1.x as i32) + (b.1.width as i32) / 2) - target_center_x;

        (a_dx * a_dx + a_dy * a_dy).cmp(&(b_dx * b_dx + b_dy * b_dy))
    });
    floating_rects
}
```

### Step 4a: Pure simulation (for preview cache) — no mutation
**File**: `crates/term-wm-core/src/window/window_manager/drag.rs`

Pure function. Returns sorted positions without touching `self`.

```rust
fn simulate_position_based_layout(
    windows: Vec<(WindowKey, Rect)>,
    anchor_key: WindowKey,
) -> Vec<(WindowKey, Rect)> {
    if windows.is_empty() {
        return windows;
    }
    let target_rect = windows
        .iter()
        .find(|(k, _)| *k == anchor_key)
        .map(|(_, r)| *r)
        .expect("anchor_key must be in windows");

    calculate_tiling_order(windows, target_rect)
}
```

### Step 4b: Destructive commit (for apply_snap) — mutates self
**File**: `crates/term-wm-core/src/window/window_manager/drag.rs`

```rust
pub(super) fn apply_position_based_layout(&mut self, anchor_key: WindowKey) {
    let windows: Vec<_> = self.regions.ids()
        .iter()
        .filter(|key| self.is_window_floating(**key))
        .map(|key| (*key, self.region(*key)))
        .collect();

    if windows.is_empty() {
        return;
    }

    let sorted = simulate_position_based_layout(windows, anchor_key);

    for (key, _) in &sorted {
        self.clear_floating_rect(*key);
    }

    let first = sorted[0];
    self.managed_layout = Some(TilingLayout::new(LayoutNode::leaf(first.0)));

    for (key, _) in &sorted[1..] {
        self.tile_window_key(*key);
    }
}
```

### Step 5: Quadrant-based snap preview cache
**File**: `crates/term-wm-core/src/window/window_manager/mod.rs`

```rust
struct SnapPreviewCache {
    positions: Vec<(WindowKey, Rect)>,
    cached_rect: Option<Rect>,
    cached_quadrant: Option<(bool, bool)>,
    dragged_key: Option<WindowKey>,
}

impl SnapPreviewCache {
    fn needs_recalc(&self, mouse_x: u16, mouse_y: u16, dragged: WindowKey) -> bool {
        let Some(rect) = self.cached_rect else {
            return true;
        };
        if self.dragged_key != Some(dragged) {
            return true;
        }
        let out_of_bounds = mouse_x < rect.x
            || mouse_x >= rect.x + rect.width
            || mouse_y < rect.y
            || mouse_y >= rect.y + rect.height;
        let cur_q = (mouse_x >= rect.x + rect.width / 2, mouse_y >= rect.y + rect.height / 2);
        out_of_bounds || self.cached_quadrant != Some(cur_q)
    }

    fn update(&mut self, mouse_x: u16, mouse_y: u16, rect: Rect, dragged: WindowKey, pos: Vec<(WindowKey, Rect)>) {
        self.cached_quadrant = Some((mouse_x >= rect.x + rect.width / 2, mouse_y >= rect.y + rect.height / 2));
        self.cached_rect = Some(rect);
        self.dragged_key = Some(dragged);
        self.positions = pos;
    }

    fn clear(&mut self) {
        self.cached_rect = None;
        self.cached_quadrant = None;
        self.dragged_key = None;
        self.positions.clear();
    }
}
```

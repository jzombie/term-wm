# Plan: Edge-Triggered Wakeups

## Problem

The reader thread fires a `Wakeup` event on every parsed byte chunk. Without parking, this floods the IPC channel with thousands of redundant messages per second, burning CPU on the main thread.

## Fix

**File:** `crates/term-wm-pty-engine/src/pty.rs`

Replace lines 525-551 (the `dirty.store` + unconditional callback block) with:

```rust
                {
                    let mut shared = shared_parser.lock().unwrap();
                    shared.process(&buf[..n]);
                }

                // Edge-triggered wakeup: only notify the main thread on the
                // false→true transition. This prevents flooding the IPC channel
                // with thousands of redundant PtyWakeup messages per second
                // when the reader is running at unthrottled speed.
                if !dirty.swap(true, Ordering::AcqRel) {
                    if let Ok(guard) = status_cb.lock()
                        && let Some(ref cb) = *guard
                    {
                        cb(crate::PtyStatus::Wakeup);
                    }
                }
```

Also move the OSC title extraction and OSC 52 clipboard handling BEFORE the dirty swap (they were between `dirty.store` and the callback in the old code). The final block should be:

```rust
                {
                    let mut shared = shared_parser.lock().unwrap();
                    shared.process(&buf[..n]);
                }

                if let Some(title) = extract_osc_title(&buf[..n])
                    && let Ok(mut guard) = pending_title.lock()
                {
                    *guard = Some(title);
                }
                // Intercept OSC 52 clipboard sequences (cross-chunk buffering).
                let tail = &history[history.len().saturating_sub(HISTORY_TAIL_LEN)..];
                if let Some(text) = osc52.push(&buf[..n], tail) {
                    let mut cb = Clipboard::new();
                    let _ = cb.set(&text);
                    if let Some(ref capture) = osc52_text {
                        *capture.lock().unwrap() = Some(text);
                    }
                }

                // Edge-triggered wakeup: only notify on false→true transition.
                if !dirty.swap(true, Ordering::AcqRel) {
                    if let Ok(guard) = status_cb.lock()
                        && let Some(ref cb) = *guard
                    {
                        cb(crate::PtyStatus::Wakeup);
                    }
                }
```

# Plan: Zero-Copy Rendering Pipeline Refactor

## Goal
Eliminate `Vec<Vec<TerminalCell>>` deep-copy. Add `process_visible_cells` to `Pane` trait — backend iterates grid, constructs transient `TerminalCell` on stack, calls UI callback. Zero heap allocations per cell, crate boundary preserved.

## Files to Modify

### 1. `crates/term-wm-pty-engine/src/pane.rs`
- **Add** `SnapshotMetadata` struct (columns, rows, cursor, default_fg, default_bg, alternate_screen, mouse, display_offset)
- **Remove** `cells: Vec<Vec<TerminalCell>>` from `TerminalSnapshot`
- **Add** to `Pane` trait:
  ```rust
  fn process_visible_cells(
      &mut self,
      columns: u16,
      rows: u16,
      cell_cb: &mut dyn FnMut(u16, u16, &TerminalCell),
      meta_cb: &mut dyn FnMut(&SnapshotMetadata),
  );
  fn is_dirty(&mut self) -> bool { false }
  ```

### 2. `crates/term-wm-pty-engine/src/pty.rs`
- **Implement** `Pty::process_visible_cells`:
  1. `self.screen()` (sync dirty/DSR)
  2. Lock `self.term`
  3. **Clamp bounds**: `columns.min(t.columns())`, `rows.min(t.screen_lines())`
  4. Iterate clamped grid, construct `TerminalCell` on stack, call `cell_cb(row, col, &cell)`
  5. Build `SnapshotMetadata`, call `meta_cb`
  6. Lock dropped after callbacks return
- **Implement** `Pty::is_dirty()` — non-destructive: `self.dirty.load(Ordering::Acquire)`
  (NOT `swap` — `screen()` is the sole consumer of the dirty flag; it controls the condvar that releases the reader thread backpressure. Stealing the flag in `is_dirty()` would deadlock the reader.)

### 3. `crates/term-wm-ui-components/src/terminal.rs` — `render_screen` (line 471)
- **Rewrite** to call `pane.process_visible_cells()`:
  - **Apply clipping bounds**: skip cells outside `visible` rectangle (`row < start_row || row >= start_row + visible.height || col < start_col || col >= start_col + visible.width`)
  - Inside cell callback: skip `wide_continuation`, map `TerminalCell` → Ratatui `buffer.cell_mut()`
  - **Accumulate row strings + byte offsets for linkifier**: track `String` + `Vec<usize>` offsets, flush with `viewport_row = current_row.saturating_sub(start_row)` (not raw `current_row`)
  - **Flush last row after closure exits** (post-loop push)
- Metadata callback: capture cursor, colors, mouse mode for cursor overlay / selection
- Keep scroll sync, drag selection, cursor overlay logic structurally unchanged

### 4. `crates/term-wm-ui-components/src/terminal.rs` — `selection_text_for_range` (line 527)

### 5. `crates/term-wm-ui-components/src/terminal.rs` — `TestPane` (line 1076)

### 6. `crates/term-session-client/src/remote_pane.rs`

### 7. `crates/term-session-client/src/lib.rs`

## Session Client Render Pattern
```rust
let mut current_row = u16::MAX;
pane.process_visible_cells(term_cols, term_rows, &mut |row, col, cell| {
    if row != current_row {
        let _ = out.queue(crossterm::cursor::MoveTo(0, row));
        current_row = row;
    }
    if cell.wide_continuation { return; }
    // encode and write cell.character
}, &mut |_meta| {});
```

## Session Client Render + Clipboard Pattern
```rust
let is_dirty = pane.is_dirty();  // capture ONCE per loop

// Drains pushes and collects clipboard texts extracted across chunks
let (raw, clipboard_texts) = pane.drain_pushes();

for text in clipboard_texts {
    let _ = clipboard.set(&text);
}

// Use captured is_dirty for render decision and pacing
if is_dirty || !raw.is_empty() {
    let mut current_row = u16::MAX;
    pane.process_visible_cells(term_cols, term_rows, &mut |row, col, cell| {
        if row != current_row {
            let _ = out.queue(crossterm::cursor::MoveTo(0, row));
            current_row = row;
        }
        if cell.wide_continuation { return; }
        // encode and write cell.character
    }, &mut |_meta| {});
}
```

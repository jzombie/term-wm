# Plan: Declarative `AppConfig` Architecture with Typestate Builder

## Critical Architecture Requirements

1. **`AppConfig` Data Store**: Create a separate `AppConfig` struct that holds all configuration and component references. `AppBuilder<M>` contains exactly `config: AppConfig` and `_mode: PhantomData<M>`.

2. **Unified Component Trait**: Create `WmComponent` trait as the lowest common denominator for all injected UI components. All panels implement `WmComponent` and are stored as `Box<dyn WmComponent>` in `AppConfig`.

3. **Embedded Validation**: `AppBuilder<Embedded>::build()` must validate `work_area_rect` is `Some`, returning `Err(ConfigError)` if missing.

4. **No Deprecation**: Remove old `WmBuilder` methods entirely.

## Phase 1: Create `WmComponent` Trait

**File:** `crates/term-wm-core/src/components/mod.rs`

```rust
/// Lowest common denominator for all WM UI components.
/// Provides type-erased storage with safe downcasting.
pub trait WmComponent: std::any::Any {
    fn as_any(&self) -> &dyn std::any::Any;
    fn as_any_mut(&mut self) -> &mut dyn std::any::Any;
}

// Blanket impl for all 'static types
impl<T: 'static> WmComponent for T {
    fn as_any(&self) -> &dyn std::any::Any { self }
    fn as_any_mut(&mut self) -> &mut dyn std::any::Any { self }
}
```

Add helper function:
```rust
pub fn wm_component_downcast<T: 'static>(comp: &dyn WmComponent) -> Option<&T> {
    comp.as_any().downcast_ref::<T>()
}

pub fn wm_component_downcast_mut<T: 'static>(comp: &mut dyn WmComponent) -> Option<&mut T> {
    comp.as_any_mut().downcast_mut::<T>()
}
```

## Phase 7: Build Phase with Validation

**File:** `crates/term-wm-core/src/config.rs`

```rust
/// Error type for configuration validation failures.
#[derive(Debug)]
pub enum ConfigError {
    /// Embedded mode requires a work_area_rect.
    MissingWorkAreaRect,
    /// AppContext must be set before building.
    MissingAppContext,
}

impl std::fmt::Display for ConfigError {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            ConfigError::MissingWorkAreaRect => write!(f, "embedded mode requires work_area_rect"),
            ConfigError::MissingAppContext => write!(f, "app_ctx must be set before building"),
        }
    }
}

impl std::error::Error for ConfigError {}

impl AppBuilder<Standalone> {
    pub fn build(self) -> Result<crate::window::WindowManager, ConfigError> {
        let app_ctx = self.app_ctx.ok_or(ConfigError::MissingAppContext)?;
        // Downcast components back to their specific types
        let top_panel = self.config.top_panel.map(|c| {
            // Downcast to TopPanel<WindowKey>
            // This will panic if the wrong type was injected
            c.into_any().downcast::<dyn TopPanel<WindowKey>>().unwrap()
        });
        // ... similar for bottom_panel, menu_overlay
        Ok(WindowManager::with_config(self.config, app_ctx, top_panel, bottom_panel, menu_overlay))
    }
}

impl AppBuilder<Embedded> {
    pub fn build(self) -> Result<crate::window::WindowManager, ConfigError> {
        // CRITICAL: Validate embedded constraints
        if self.config.work_area_rect.is_none() {
            return Err(ConfigError::MissingWorkAreaRect);
        }

        let app_ctx = self.app_ctx.ok_or(ConfigError::MissingAppContext)?;
        // Downcast and build...
        Ok(WindowManager::with_config(self.config, app_ctx, top_panel, bottom_panel, menu_overlay))
    }
}
```

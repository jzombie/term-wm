# Plan: Extract WM-specific components into `term-wm-sys-ui-components`

## Goal
Create a new crate `term-wm-sys-ui-components` that holds window-manager-specific components currently mixed into `term-wm-ui-components`. This separates reusable UI widgets from WM infrastructure.

## Components to move (into new crate)

| Component | Current location | Type |
|-----------|-----------------|------|
| `WmMenuOverlay<R>` | `term-wm-ui-components/src/menu_overlay.rs` | Menu overlay implementing `MenuOverlay` trait |
| `DebugLogComponent` | `term-wm-ui-components/src/sys/debug_log.rs` | System window (debug log) |
| `HelpOverlayComponent` | `term-wm-ui-components/src/sys/help_overlay.rs` | System overlay (help) |
| `KeybindingOverlayComponent` | `term-wm-ui-components/src/sys/keybinding_overlay.rs` | System overlay (keybindings) |
| Sys module container | `term-wm-ui-components/src/sys/mod.rs` | Module declarations |
| `assets/help.md` | `term-wm-ui-components/assets/help.md` | Embedded help content |

## New crate structure

```
crates/term-wm-sys-ui-components/
├── Cargo.toml
├── assets/
│   └── help.md              # Moved from term-wm-ui-components/assets/
└── src/
    ├── lib.rs               # Re-exports all public items
    ├── menu_overlay.rs      # WmMenuOverlay<R> (moved, imports updated)
    └── sys/
        ├── mod.rs           # pub mod debug_log; pub mod help_overlay; pub mod keybinding_overlay;
        ├── debug_log.rs     # DebugLogComponent (moved, imports updated)
        ├── help_overlay.rs  # HelpOverlayComponent (moved, imports updated)
        └── keybinding_overlay.rs  # KeybindingOverlayComponent (moved, imports updated)
```

## Dependencies (`crates/term-wm-sys-ui-components/Cargo.toml`)
- `crossterm` (workspace) — event handling
- `ratatui` (workspace) — rendering
- `term-wm-core` (path = "../term-wm-core") — traits, UiFrame, theme, keybindings
- `term-wm-ui-components` (path = "../term-wm-ui-components") — reusable widgets (ScrollView, Dialog, Menu, etc.)
- `webbrowser` (workspace) — used by HelpOverlayComponent for link handling

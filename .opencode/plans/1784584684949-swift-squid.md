# Plan: Monocle Mode Command Palette & Chrome Fixes

## Problem Statement

Two issues in monocle mode:

1. **"Close Window" missing from command palette**: In `wm_menu_items()` (`crates/term-wm-core/src/window/window_manager/mod.rs:2302-2338`), when `is_monocle` is true, only "Switch to:" items are added. The "Close Window" command is in the `else` branch and never appears in monocle mode.

2. **Minimize/Maximize chrome buttons visible in monocle mode**: In `render_window()` (`crates/term-wm-console/src/draw_plan_renderer.rs:907-949`), all 4 buttons (Close, Max, Min, DirectMode) are rendered unconditionally. No monocle check exists.

The user wants to **centralize the gating** so monocle-specific visibility is defined once and used by both the command palette menu builder and the chrome renderer.

## Approach

### Step 1: Update `wm_menu_items()` to include "Close Window" in monocle mode

In `crates/term-wm-core/src/window/window_manager/mod.rs:2302-2338`:

Move "Close Window" **before** the `if is_monocle` branch so it appears in both modes. Keep Maximize, Minimize, and Toggle Direct Mode only in the non-monocle branch.

### Step 2: Add `is_monocle` to `ChromeCtx` and filter buttons

In `crates/term-wm-console/src/draw_plan_renderer.rs`:

1. Add `is_monocle: bool` field to `ChromeCtx` (line 22-29)
2. In `render_window()` (line 907-949), conditionally skip Min/Max buttons when `is_monocle` is true. Close and DirectMode remain visible.
3. In `register_window_chrome_hitboxes()` (line 43-141), add `is_monocle: bool` parameter and skip MinimizeButton/MaximizeButton hitbox registration when true.

### Step 3: Update all call sites

**`ChromeCtx` construction sites:**
- `src/lib.rs:130` — add `is_monocle: wm.is_monocle()`
- `crates/term-wm-console/src/draw_plan_renderer.rs:1569` (test) — add `is_monocle: false`
- `crates/term-wm-console/src/draw_plan_renderer.rs:1659` (test) — add `is_monocle: false`

**`register_window_chrome_hitboxes` call site:**
- `crates/term-wm-console/src/draw_plan_renderer.rs:167` — pass `ctx.is_monocle`
